# BioSage AI

BioSage AI is a biotechnology-based Android application developed using Kotlin and Android Studio.

## Features
- Biotechnology calculator
- Molarity and Molality calculations
- Simple Android UI
- AI-inspired healthcare concept
- User-friendly design

## Technologies Used
- Kotlin
- Android Studio
- XML
- Android Emulator

## Project Objective
The objective of this project is to develop a beginner-friendly biotechnology Android application integrating AI concepts and biotech utilities.

## Future Scope
- AI chatbot integration
- DNA sequence analysis
- Bioinformatics modules
- Cloud database support
